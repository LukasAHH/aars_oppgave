Maze escape

Navigate the maze filled with boxes. Solve puzzles and build the key by collecting fragments to advance!

Player :
    the playable character, the one you move around and control

Box :
    Boxes than act as walls but can be pushed if path is not blocked

Key fragments :
    Collecting all of these will change the color of the door and allow you to pass through and win

Golden coins :
    Extra collectibles for extra gratification or adding to some score

Empty = 0
Wall = 1
Player = 2
Coin = 3
Fragment = 4

Exit door = 9
